# Jinguzhou 0.3.0 Agent Ecosystem Release Plan

This release turns Jinguzhou into a small agent tool firewall layer for common
tool-calling runtimes.

## Scope

- adapter foundation
- JSONPath-like extractor completion
- file, network, and database policy packs
- MCP adapter helpers
- LangChain-style tool middleware
- LlamaIndex adapter helpers
- OpenAI Agents adapter helpers
- runnable examples and release validation

## Acceptance

- all v0.3 examples run offline
- tool calls normalize into `NormalizedToolCall`
- policy packs block the expected file, network, and database examples
- release validation passes

## Follow-Up After 0.3.0

- package-specific cookbooks for real MCP servers and agent frameworks
- deeper approval flow integration for tool middleware
- audit display improvements for normalized tool calls
